### 차별화 #1 — 딸깍 전용 CLAUDE.md (최대 차별점)

**원리:**
스타트업 생성 시 `/workspace/{id}/CLAUDE.md` 자동 생성.
Claude Code가 이 디렉토리에서 실행되면 CLAUDE.md를 자동으로 읽음.
→ 우리 규칙이 Claude Code 동작에 자동 적용.

**규칙 내용:**
1. `dalkkak.json` 필수 생성 (start 명령어 + port)
2. 서버는 0.0.0.0 바인딩 (Docker 네트워킹 필수)
3. `/health` 엔드포인트 필수 (헬스체크용)
4. 환경변수는 .env 파일 사용
5. requirements.txt / package.json 필수

**왜 차별화인가:**
- 다른 서비스 (Devin, Cursor, Codex): 그냥 Claude/GPT 띄워줌
- DalkkakAI: **규약이 내장된** Claude Code를 제공
- 결과물이 항상 Test 버튼으로 실행 가능한 형태로 나옴
- 유저가 규칙을 몰라도 자동으로 "배포 가능한 코드"가 생성됨

**확장 가능성:**
- 플랜별로 다른 CLAUDE.md (Free=기본, Scale=보안+모니터링+성능최적화)
- 도메인별 CLAUDE.md (제조=센서연동규칙, 금융=보안규칙, 의료=HIPAA규칙)
- 유저 커스텀 CLAUDE.md (고급 유저가 자기만의 규칙 추가)

### 차별화 #2 — 두 가지 모드 (Terminal + Auto AI)

**Terminal 모드:**

**같은 UI, 같은 Test 버튼, 같은 결과.** 모드만 다름.

### 차별화 #4 — 코딩 너머 전체 운영

```
다른 서비스: 아이디어 → 코드 (끝)
DalkkakAI:   아이디어 → 코드 → 테스트 → 배포 → 마케팅 → 결제 → 지원 → 분석
```

Phase 1에서 코딩+테스트+배포까지.
Phase 2~3에서 마케팅+결제+지원+분석 추가.

### 차별화 #6 — CLAUDE.md 기반 Agent 고도화 시스템

**핵심 인사이트:**
같은 Claude Code인데 CLAUDE.md 하나로 주니어 → 시니어급 코드가 나옴.
Devin/Cursor는 이런 거 없음 — 그냥 LLM 띄워주는 것.
DalkkakAI는 **규약이 내장된** AI를 제공.

**CLAUDE.md로 제어 가능한 것들:**
```
기본: dalkkak.json, 0.0.0.0, /health, .env
고도화:
  - ReAct 패턴 (Thought → Action → Observation 루프)
  - TDD (테스트 먼저 작성)
  - 에러 발생 시 자동 재시도 (최대 3회)
  - 파일 300줄 넘으면 자동 분리
  - OWASP Top 10 보안 체크
  - 성능 최적화 패턴 적용
  - 아키텍처 결정마다 이유 주석
  - CI/CD 설정 자동 생성
  - 모니터링 자동 설정
```

**플랜별 차별화:**
```
Free:
  기본 규칙 (dalkkak.json, /health, 0.0.0.0)

Starter ($29):
  + TDD (테스트 먼저)
  + 에러 핸들링 강제
  + README 자동 생성

Growth ($99):
  + ReAct 패턴 사고
  + 보안 체크 (OWASP)
  + CI/CD 설정 자동 생성
  + 코드 리뷰 체크리스트

Scale ($299):
  + 성능 최적화 패턴
  + 모니터링 자동 설정
  + 마이크로서비스 분리 가이드
  + 자동 스케일링 설정
```

**도메인별 특화 CLAUDE.md:**
```
제조: 센서 연동 규칙, 실시간 데이터 처리 패턴, 알람 시스템
금융: 보안 규칙 (PCI DSS), 트랜잭션 관리, 감사 로그
의료: HIPAA 규칙, PHI 데이터 암호화, 접근 제어
커머스: 결제 연동, 재고 관리, 배송 추적
```

**중요 설계 결정: 강제가 아닌 기본값**
```
문제: 개발자는 강제 규칙을 싫어함
해결: CLAUDE.md는 "기본값"이지 "강제"가 아님

비개발자: 기본값 그대로 사용 → 규칙을 몰라도 좋은 코드 나옴
개발자:   Settings에서 규칙 ON/OFF 토글
          커스텀 규칙 추가 가능
          CLAUDE.md 직접 편집 가능
          아예 삭제도 가능

이렇게 하면:
  비개발자 = 보호받음 (기본 규칙이 품질 보장)
  개발자 = 자유로움 (원하는 대로 커스텀)
  둘 다 만족
```
